# Backend 

This is a UI-related backend that is compatible with the `ARElight-server`.

Powered by d3-js.

**See:** https://github.com/nicolay-r/ARElight-server

## Data Layout

```
output/
├── description/
    └── ...         // graph descriptions in JSON.
├── force/
    └── ...         // force graphs in JSON.
├── radial/
    └── ...         // radial graphs in JSON.
└── index.html      // main HTML demo page. 
```
